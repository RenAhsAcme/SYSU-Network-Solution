## 报告漏洞

若涉及安全漏洞，在没有预先授权的情况下，请您务必**不要公开**讨论它们，**不要提交 Issue，Pull Request，Discussion等！**

您可以通过 GitHub 的**私密漏洞报告**功能向我提交漏洞，我会及时处理它们。

您也可以通过邮件向我报告漏洞，但请**务必完成对邮件的加密**。您可以使用 PGP 密钥对邮件进行加密。以下是密钥的相关信息：

在您报告漏洞以后，我会尽快给予回复并告知具体完成的修复时间，我们可能需要更多具体协商往来。协商往来期间，所有邮件仍需加密，且不接受使用其他不支持端到端加密的、不能直接在中国大陆直接使用的联系方式。
